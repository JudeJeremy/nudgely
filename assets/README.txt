# Assets Directory

This directory contains static assets for the Nudgely app.

## Required Assets

The following assets should be placed in this directory:

- `icon.png`: App icon (1024x1024 px)
- `splash.png`: Splash screen image (1242x2436 px)
- `adaptive-icon.png`: Android adaptive icon foreground (1024x1024 px)
- `favicon.png`: Web favicon (196x196 px)
- `notification-icon.png`: Notification icon (96x96 px)
- `notification-sound.wav`: Notification sound

## Asset Guidelines

- All images should be in PNG format (except for sounds)
- Use transparent backgrounds where appropriate
- Follow the size guidelines for each asset
- Use the app's primary color (#0a7ea4) for backgrounds where needed
